Task 1: Data Cleaning and Preprocessing

 Objective
To clean and preprocess an E-commerce Sales dataset by handling missing data, removing duplicates, and standardizing columns.

 Tools Used
- Microsoft Excel
  
Steps Performed
1. Removed duplicate rows  
2. Handled missing values (filled numeric nulls with median; dropped incomplete rows)  
3. Standardized text fields (e.g., Region, Category, Ship Mode)  
4. Converted dates to consistent format (`dd-mm-yyyy`)  
5. Renamed column headers to lowercase with underscores  
6. Checked and fixed data types (int, float, datetime)

 Deliverables
- Cleaned Dataset (Excel)
